== Description ==

Added new feature and fixed some problem when enable debug mode

Special thanks to https://github.com/kanmanus/mourn.js for the awesome jQuery library

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/hiro-greymode` directory, or install the plugin through the WordPress plugins screen directly

2. Activate the plugin through the 'Plugins' screen in WordPress

3. Use the Grey Mode Name screen to configure the plugin

4. That's it!

== Frequently Asked Questions ==

= How it's work? =

This plugin can change color in whole page to grey color, and also have toggle box to change to normal color
